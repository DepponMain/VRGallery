# VR_Gallery

#### 项目比较大，东西有点多，慢慢抽取，持续更新...（本人Github上第一个比较大的项目。望多多支持！考虑到大家pod比较麻烦，为了方便运行，所以就将pod一并提交上去了）

## 小目标
代码要写得优雅、做好封装与抽取，每个.m文件都不能超过三百行代码！

## ☆注意☆
考虑到部分同学没有VR图片，我上传了几张低质量的，放在 PicSource 文件夹里，可以传到自己的手机里，或者拖到模拟器里（PS：每张图片多复制几张，可以更清楚的查看效果）
